# Как анимировать объекты в Tilda

## Способ 1: Через код (ID или класс блока)

### Логотип или один блок по ID

1. Откройте файл **`logo-entrance.html`** — скопируйте из него весь код (теги `<style>...</style>`).
2. В Tilda: **Настройки страницы** → **Дополнительный код** → вставьте код в поле **«Перед закрывающим тегом </body>»**.
3. В редакторе **выделите блок**, в котором логотип (или нужный объект).
4. Откройте **настройки блока** (шестерёнка) → найдите поле **«ID блока»**.
5. Введите: **`dafull-logo`** (латиницей, без решётки). Сохраните блок.
6. **Опубликуйте** страницу — при загрузке блок плавно появится (снизу вверх + лёгкое увеличение).

### Несколько блоков по классу

1. Скопируйте код из файла **`animate-by-class.html`** в **«Перед закрывающим тегом </body>»** (как выше).
2. Для каждого блока, который нужно анимировать: настройки блока → поле **«Класс для блока»** (или «CSS class») → добавьте один из классов:
   - **`dafull-animate-in`** — появление снизу + увеличение
   - **`dafull-animate-fade`** — только плавное проявление
   - **`dafull-animate-up`** — выезд снизу вверх
3. Опубликуйте страницу.

Если в настройках блока нет поля «Класс для блока», используйте способ с ID (для одного блока задайте ID `dafull-logo`).

## Способ 2: Встроенная анимация Tilda (без кода)

### Вариант 1: Обычный блок (не Zero Block)

Если логотип добавлен обычным блоком «Картинка» или «Колонки с картинкой»:

1. **Кликните по блоку** с логотипом (один клик — выделится весь блок).
2. Откройте **настройки блока** (иконка шестерёнки справа или двойной клик по блоку).
3. Найдите раздел **«Анимация»** или **«Эффект появления»**.
4. Выберите один из вариантов:
   - **Прозрачность (увеличение)** — лого плавно увеличивается из центра (похоже на «красивое появление»).
   - **Прозрачность (снизу)** — выезжает снизу вверх.
   - **Прозрачность (сверху)** — сверху вниз.
   - **Прозрачность** — простое появление без движения.
5. Сохраните и **опубликуйте** страницу.

---

### Вариант 2: Zero Block

Если логотип находится внутри Zero Block:

1. Откройте страницу в редакторе Tilda.
2. **Дважды кликните** по Zero Block, чтобы войти внутрь него.
3. **Кликните один раз** по элементу с логотипом (картинке или контейнеру).
4. В правой панели найдите **«Basic Animation»** / **«Базовая анимация»** или вкладку с эффектами элемента.
5. В поле **«Animation»** / **«Анимация»** выберите:
   - **Zoom In** — появление с увеличением (красиво для лого).
   - **Fade In Up** — появление снизу вверх.
   - **Fade In** — простое появление.
6. При необходимости задайте **Delay** (задержку в секундах), чтобы лого появился чуть позже.
7. Выйдите из Zero Block (клик вне блока или кнопка «Назад») и **опубликуйте** страницу.

---

### Вариант 3: Пошаговая анимация (Step by Step) в Zero Block

Для более сложного эффекта (например, лого масштабируется и потом останавливается):

1. В Zero Block выделите элемент с логотипом.
2. Найдите **«Step by Step Animation»** / **«Пошаговая анимация»**.
3. Нажмите **«Add trigger»** / **«Добавить триггер»**.
4. Выберите **«Block on Screen»** или **«Element on Screen»** (анимация при появлении блока/элемента на экране).
5. Добавьте шаги анимации (например: шаг 1 — opacity 0, scale 0.9; шаг 2 — opacity 1, scale 1).
6. Сохраните и опубликуйте.

---

## Если не видите пункт «Анимация»

- Убедитесь, что выделен именно **блок** или **элемент внутри Zero Block**, а не вся страница.
- Некоторые старые блоки могут не поддерживать анимацию — попробуйте заменить блок на актуальный из каталога Tilda.
- В **настройках страницы** проверьте, что не отключена микро-анимация страницы (раздел «Настройки отображения» / «Дополнительно»).

---

Итог: анимацию можно задать **через код** (Способ 1) или **встроенными средствами Tilda** (Способ 2).
