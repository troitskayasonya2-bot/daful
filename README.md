Код блоков для вставки в Tilda (Zero Block / HTML-блок).

## Цвета
- **#FF6300** — акцентный оранжевый (кнопки, акценты)
- **#023A11** — зелёный (лого, микрометки)
- **#13161F** — тёмный фон лого / заголовки
- **#FAFAFA / #fff** — светлый фон

## Квиз и попап
- **Чтобы квиз показывался в попапе:** в Tilda откройте блок с квизом/формой → настройки блока (иконка шестерёнки) → найдите поле **«ID»** / **«Block ID»** / **«Record ID»** и впишите **rec1947644431** (без пробелов). Сохраните. Тогда скрипт найдёт блок и вставит его копию в попап.
- Все кнопки/ссылки с `href="#rec1947644431"` открывают квиз во всплывающем окне. Закрытие: крестик, клик по фону или Escape.
- Если квиз в попапе не появился — проверьте, что у блока с формой/квизом действительно указан ID `rec1947644431`.

## Вёрстка (ширина и кроссбраузерность)
- Все блоки должны **фиксироваться по ширине экрана**: `width: 100%`, `max-width: 100%`, `overflow-x: hidden` у секции; `box-sizing: border-box` у контейнеров.
- У grid/flex-ячеек задавать `min-width: 0`, чтобы контент не раздувал сетку и не вызывал горизонтальный скролл.
- Цель: одинаковое и корректное отображение на всех устройствах и во всех браузерах без «съезжания».

## Как вставлять в Tilda
- Создать блок **Zero Block** или **HTML**.
- Вставить содержимое соответствующего файла (весь HTML вместе с `<style>...</style>`).
- При необходимости заменить ссылку на изображение на своё (загрузить в медиабиблиотеку Tilda и подставить URL).
